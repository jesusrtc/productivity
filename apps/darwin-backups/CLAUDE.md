# Darwin Backups

Scripts to download code files (.ipynb, .py, .swb) from Darwin.

## Setup

```bash
./setup.sh
```

## Usage

```bash
./darwin-backups <darwin_path_or_url>
```

(The `darwin-backups` wrapper is generated by `setup.sh` and pinned to
the local `venv/`. You can also call `venv/bin/python3 download.py`
directly.)

## Key details

- Downloads go to `downloads/` preserving the Darwin folder structure
- Notebook outputs are stripped to keep files clean
- The source `download.py` / `download-all.py` use `#!/usr/bin/env python3`
  so the repo is portable; `setup.sh` generates a `darwin-backups`
  wrapper that binds them to the local `venv/bin/python3`.
- Auth token comes from datavault or captain cache; if expired run `captain setup darwin`
